Quien eres tú Para alardear y presumir / mejor pregúntale a ella ¿Quién es el hombre que la eleva a las estrellas? Quien de los 2 la hizo sentirse más mujer

  • alardear = presumir - хвастаться

Quien eres tú para retarme Y hacer de ella una competencia Así aseguro que es infame lo que alegas

  • retar - бросать вызов
  • competencia - конкуренция
  • Así - так
  • alegar = decir - утверждать
  • infamé - с дурной славой
