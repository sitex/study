Rodrigues

p 178 - Gramatica

Rodrigues

p 179 - 1

  • asistir - посещать

Modismos

  • muy amable - очень любезно (спасибо)
  • con permiso - пожалуйста (формально, просят разрешение)
  • en seguida = de una vez - сейчас же
  • caer bien / mal - нравится (не нравится) (кто-то)
  • no cae bien / cae mal (comida) - тошнит от еды

  • para ser franco / sincero / honesto - честно говоря

  • faltar al respeto a (alguien) - проявлять неуважение (грубить, домогаться)

  • mas vale … que … - лучше

  • mas te vale - смотри у меня

  • ocurirse (a) (alguien) … - приходить в голову (что-то)

    • se (me/te/le) ocurio

  • es hora de (hacer algo) - пора

  • (con) respecto a … = al respecto de … - относительно (чего-то)

  • darse cuenta de (…) - понять / осознать (что-то)

  • tener la razon - быть правым

  • perder la razon - сходить с ума

  • dejar en paz - оставить в покое

  1. понять /la mesera/ нравится // лучше - дать чаевые
  2. честно говоря // пора // сейчас же
  3. пожалуйста // сейчас же // оставить в покое
  • como se te ocurio - как тебе пришло в голову
  1. Estaba trabajando cuando se me ocuririo hacer la tarea.
  2. Romeo y Julieta [se caían bien] desde que se conocieron.
  3. Madrastra de Cenicienta [faltaba al respeto] a Cenicienta y la hacía trabajar mucho.
  4. Después de estudiar [me di cuenta de] lo que decía el profesor.
  5. (tener la razon) Cuando me di cuenta que [tenían razón], les agradecí.

Modismos 2

  • guardar distancia
  • estar en (mi|su|tu) sano juicio - быть в здравом уме
  • contra (mi|tu|su) voluntad - против воли
  • no tener ningún inconveniente - без проблем
  • entrar en possession - вступить в права собственности

  • no hacer nada mas que … - ничего не делать кроме … (только и делать что …)

  • (a alguien) (me|te|le) cuesto trabajo - быть сложным

Alumno

p 129 - 7

  • dibujas animadas = caricaturas - мультики

  • que tan seguido? - как часто?
